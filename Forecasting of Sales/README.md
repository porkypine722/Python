# 📈 Sales Forecasting with XGBoost

Predicting future sales from historical sales data using **XGBRegressor**, with exploratory data analysis, feature engineering, and model evaluation.

---

## 🔍 Overview

This project builds a machine learning pipeline to forecast sales figures based on historical sales data. The goal is to produce accurate short-term sales predictions that can help with inventory planning and business decision-making.

---

## ⚙️ Approach

1. **Data Preprocessing** — handling missing values, outlier treatment, encoding categorical variables
2. **Feature Engineering** — lag features, rolling averages, date-based features (month, day of week, quarter)
3. **Modelling** — trained an `XGBRegressor` with hyperparameter tuning via `GridSearchCV`
4. **Evaluation** — assessed performance using RMSE

---


## 🛠️ Tech Stack

- Python 3.x
- scikit-learn
- pandas, NumPy
- Matplotlib, Seaborn
