# 🥗 Cost of a Healthy Diet by Country — Data Visualisation

Exploring the relationship between the cost of a healthy diet and the cost of vegetables and fruits across countries, using **pandas** and **Matplotlib/Seaborn**.

---

## 🔍 Overview

Access to affordable, healthy food varies widely across the world. This project analyses a country-level dataset containing the cost of vegetables, fruits, and a complete healthy diet to uncover how these factors relate to each other — visualised through clear, interpretable scatter plots.

---

## ⚙️ Approach

1. **Data Loading** — loaded the CSV using `pandas` and inspected for missing values
2. **Data Cleaning** — dropped null entries, standardised column names
3. **EDA** — summary statistics and distribution checks for each cost variable
4. **Visualisation** — produced scatter plots using Matplotlib/Seaborn with country-level annotations

---

## 🛠️ Tech Stack

- Python 3.x
- pandas
- Matplotlib
- Seaborn
